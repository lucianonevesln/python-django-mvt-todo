# Projeto: Frontend/Backend em Django

## Objetivo:

### Entender na prática como funciona a interação entre JavaScript/AJAX e Django através de estruturas semi-independentes que representam uma lista de atividades.

## Tecnologias Utilizadas:

### - Python;

### - JavaScript;

### - SQLite3;

### - AJAX (Fetch);

### - Bootstrap;

### - CSS;

### - HTML;
